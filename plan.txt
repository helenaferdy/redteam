# 3-Month Internal Red Team Capability Development

**Objective**

Build an enterprise-ready Internal Red Team capable of delivering penetration testing and Red Team engagements using **CyberStrike** as the primary offensive platform while understanding the underlying attack techniques.

---

# Week 2 — Reconnaissance & Asset Discovery

## Goal

Produce a professional Reconnaissance Report.

### Manual Foundation

Run

```bash
nmap -sV -sC <target>
```

Learn

- Port Scanning
- Service Detection
- Version Detection

Run

```bash
rustscan -a <target>
```

Compare results.

### CyberStrike

**Agent**

`cyberstrike`

Prompt

```
Perform reconnaissance against the target network.

Identify:

- Live hosts
- Open ports
- Running services
- Technologies
- Initial attack vectors

Generate an executive and technical summary.
```

Engineer validates findings.

### Deliverables

- Recon Report
- Recon Playbook
- Prompt Library v1

# Week 5 — AI-Assisted Vulnerability Validation

## Goal

Understand how CyberStrike validates vulnerabilities.

### Manual Foundation

Select ONE vulnerability.

Example

SQL Injection.

Demonstrate manually.

Understand why it works.

### CyberStrike

**Agent**

`web-application`

HackBrowser

Autonomous Mode

Prompt

```
Login using supplied credentials.

Automatically crawl the application.

Capture every endpoint.

Execute all available Proxy Testers.

Only report vulnerabilities confirmed through reproducible evidence.

Generate screenshots and proof-of-concept requests.
```

Expected CyberStrike Features

- HackBrowser
- IDOR Tester
- Authentication Tester
- Authorization Tester
- Injection Tester
- Business Logic Tester
- SSRF Tester
- File Attack Tester

Engineer validates every finding.

### Deliverables

- Vulnerability Assessment Report
- Evidence Pack
- Validation SOP
